Autodesk Revit

AutoDesk Revit has a powerful API to allow software developers to make add-ins which communicate with the Revit Model. The add-ins can save designers a lot of time and can improve the quality of the model. If you are interested in an add-in, please feel free to ask me about the possibilities (tom@tnux.net).

I’ve made several applications, one application is available through the Revit App Store. This application allows the user to auto number parking lots. Read more about my activity on Revit, here. The paid version allows the user to auto number any element with a mark value.

All plugins below are for sale, please contact me for a trial version or licensing details at tom@tnux.net.

Available add-ins:

  • Revit Angle Validation
  • Revit Auto Number Add-in
  • Revit Export Add-in
  • Revit Save Selection
  • Revit Tag By Filter
  • Revit Type Name Manager
  • Revit View Template Manager
  • Revit Wall Finish Add-in
